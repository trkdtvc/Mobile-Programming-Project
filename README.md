# Mobile Programming Project

A mobile-focused application project developed as part of mobile programming coursework. The project demonstrates core concepts of mobile app development, including UI structure, user interaction, and application logic.

## Features
- Mobile-oriented user interface
- Interactive components and user input handling
- Structured project layout
- Focus on core mobile development concepts

## Tech Stack
- Java
- Android (Android Studio)

## Purpose
Built as a practice and coursework project to gain hands-on experience with mobile application development fundamentals, UI design, and event-driven programming on the Android platform.
